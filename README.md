Titanic Survival Prediction using Neural Network
Overview:
This project predicts whether a passenger survived the Titanic disaster using a neural network model. The model learns patterns from passenger data and classifies survival outcomes.

Dataset:

The dataset used in this project is the Kaggle Titanic dataset.

Features Used:
The model is trained using three main features:

Passenger Class (Pclass)
Age
Sex

These features were selected to keep the model simple while still capturing important survival factors.
Model
A neural network model is used to classify whether a passenger survived or not. The dataset is preprocessed by handling missing values and encoding categorical variables before training the model.

Project Pipeline
Load Titanic dataset
Select important features (Pclass, Age, Sex)
Data preprocessing and encoding
Train neural network model
Evaluate model predictions
Goal

The goal of this project is to apply deep learning techniques to predict passenger survival based on basic passenger information.
